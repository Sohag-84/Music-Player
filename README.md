# Music Player

Flutter Music player app

### ✍🏽The main features of our app are:
🎵 All mp3 audio list </br>
🎵 Play song</br>
🎵 Show song name</br>
🎵 Show Artist name</br>
🎵 Pause song</br>
🎵 Go Preveous song</br>
🎵 Go Next song</br>
